# Zenture Analyst

App pronto para a Zenture analisar treinos/campeonatos de Free Fire.

## Login padrão
- Usuário: `zenture`
- Senha: `ZNT2026`

Você pode alterar no arquivo `.env.local`.

## Rodar no notebook
1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. Abra o terminal e rode:

```bash
npm install
npm run dev
```

4. Abra no navegador:

```bash
http://localhost:3000
```

## Subir grátis na Vercel
1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta.
3. Entre na Vercel.
4. Clique em Add New > Project.
5. Importe o repositório.
6. Clique em Deploy.

## O que está pronto
- Login e senha.
- Dashboard.
- Cadastro de squads.
- Cadastro de partidas.
- Ranking automático.
- Upload de print.
- OCR gratuito com Tesseract.js.
- Correção manual do resultado lido.
- Histórico salvo no navegador com localStorage.
- Design preto e azul cobalto da Zenture.

## Observação sobre IA/OCR
A leitura é gratuita e roda no navegador. Ela consegue ler textos de prints, mas prints com baixa qualidade podem exigir correção manual.
